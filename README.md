# Ingeniería II — 2026

Repositorio oficial de la cátedra **Ingeniería II**, correspondiente al **2.º año** de la **Tecnicatura Superior en Desarrollo de Software**.

## 📚 Descripción

Este repositorio será utilizado a lo largo del cursado de la materia para alojar los proyectos, prácticas y trabajos prácticos desarrollados durante el año lectivo 2026. El objetivo es aplicar y profundizar conocimientos en el desarrollo de aplicaciones web mediante tecnologías modernas del frontend.

## 🛠️ Tecnologías utilizadas

| Tecnología | Descripción |
|------------|-------------|
| ![HTML5](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white) **HTML5** | Estructura y semántica de las páginas web |
| ![CSS3](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white) **CSS3** | Estilos, diseño y presentación visual |
| ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black) **JavaScript** | Lógica, interactividad y dinamismo en el cliente |
| ![Bootstrap](https://img.shields.io/badge/Bootstrap-7952B3?logo=bootstrap&logoColor=white) **Bootstrap** | Framework CSS para diseño responsive y componentes UI |

## 🎯 Objetivos de la materia

- Comprender y aplicar los fundamentos del desarrollo web frontend.
- Construir interfaces de usuario responsivas y accesibles.
- Integrar JavaScript para la manipulación del DOM y el manejo de eventos.
- Utilizar Bootstrap para agilizar el diseño y la maquetación.
- Fomentar buenas prácticas de desarrollo y trabajo colaborativo con Git y GitHub.
