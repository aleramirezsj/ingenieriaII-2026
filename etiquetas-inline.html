Ejemplos de etiquetas inline

En desarrollo web, las etiquetas inline permiten dar formato al texto sin cortar el flujo del párrafo. Por ejemplo, este texto es importante y este texto tiene énfasis.

Podemos destacar una idea principal, agregar una nota con texto de tamaño menor o subrayar una expresión.

También se puede escribir un precio anterior como $2500 y uno nuevo como $1999. En química vemos H2O y en matemática podemos escribir x2.

Si queremos mostrar código, usamos <h1>Hola</h1>. Para indicar una combinación de teclas podemos escribir Ctrl + S. Una abreviatura útil es HTML.

Una cita corta podría ser: Aprender haciendo es una gran forma de crecer. Esta frase puede aparecer en una clase de desarrollo web.
