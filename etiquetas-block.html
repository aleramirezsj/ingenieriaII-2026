Introducción

Las etiquetas block ocupan todo el ancho disponible y normalmente comienzan en una nueva línea. Son ideales para estructurar el contenido principal de una página.

En esta página se muestran varios ejemplos útiles para aprender a organizar información en HTML.

Contenedores y contenido

Artículo de ejemplo

Un artículo puede contener información completa y con sentido propio, como una noticia o una publicación.

Contenedor div

La etiqueta div es muy utilizada para agrupar elementos y aplicar estilos o estructura visual.

Dato adicional

Un bloque aside puede usarse para notas, recordatorios, publicidad o información secundaria.
